# Photo Sorter

Desktop app for sorting photos into a year/month/day folder structure. Built with Tauri, React, and TypeScript.

> **Disclaimer:** This is 100% vibe coded and meant for my own personal use. There are no tests, no guarantees, and no promises that it won't do something dumb with your files. Use at your own risk — maybe back stuff up first.

## Dev

Requires [Bun](https://bun.sh), [Rust](https://rustup.rs), and [Tauri prerequisites](https://v2.tauri.app/start/prerequisites/).

```bash
bun install
bun run tauri dev
```

Build:

```bash
bun run tauri build
```
